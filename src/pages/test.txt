PAGES FOLDER
=============

Purpose: Page components (full page views)

Use for:
- Home page
- Product page
- User profile page
- Checkout page
- Error pages (404, 500)
- Login/Register pages
- Settings pages
- Dashboard page

Structure suggestion:
- Create subfolders for different page sections
- Each page can have its own folder with components and styles

Example files to create:
- HomePage/HomePage.jsx
- ProductPage/ProductPage.jsx
- ProfilePage/ProfilePage.jsx
- CheckoutPage/CheckoutPage.jsx
- NotFound/NotFound.jsx
- LoginPage/LoginPage.jsx
- DashboardPage/DashboardPage.jsx
